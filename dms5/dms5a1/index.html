1. What genre of creative software do you believe it belongs to?

This creative software belongs to a digital drawing tool genre.

2. List three other examples of software in this genre

Other similar softwares in this genre could be:

  • Microsoft Paint
  • Nintendo Flipnote Studio
  • Clip Studio paint

3. List the program's interactive features/mechanics (one sentence max explanation per feature/mechanic)

The program features:

  • A full suite of digital drawing brushes, such as ink pen, pen, paintbrush, scatter brushes, coloured markers and an eraser.
  • A canvas that allows for the user to freely doodle using one of the listed brushes, and makes the resulting artwork “wiggle”
  • Additional customisation settings give the user the freedom to manipulate the size, arrangement, crop, colour and shape of the artwork and the program as a whole.
  • Sound effects that give the user an audio feedback whenever something on the program is interacted with.
  • The option to save the working file to computer, import images or export a gif.

4. What features/mechanics indicate its genre by aligning with its conventions?

The features enabling users to draw, edit, erase and delete artworks aligns with the genre of Digital drawing tool as other tools also primarily allow it’s users to manipulate brushes, colours and basic editing to complete an artwork. The main difference between a Digital drawing tool and a full on digital editing tool is the lack of advanced features that users might find in softwares like photoshop.

5. What features/mechanics are novel for its genre?

A novel feature for this program in it’s genre is the “wiggle” that is added to every brushstroke. Most digital drawing tools produce static, non-moving artworks, however, this program features a “wiggle” mechanic that sets it apart from other programs.

6. What features/mechanics common to its genre are missing or might be added?

One missing feature in this program that is commonly found in other softwares in it’s genre is the ability to work with layers. Most digital drawing tools contains layer features that allows for the user to work within multiple layers, however, this program does not.

7. What context, community or niche is it primarily aimed at and what indicates this?

This program is primarily aimed at creatives who wants to create casual, simple artworks without the complexity of a full suite drawing software, easily exporting it to be shared online or elsewhere.

8. What did you find annoying and/or boring about interacting with it?

One downside to the program is that the undo feature only undos a single action, and the only way to rid previous mistakes if it’s overwritten is to use the eraser. And without a layer system, it’s difficult to organise the artworks and correct mistakes.

9. What did you find enjoyable about interacting with it?

I enjoy the endless possibilities that this program allows, with the combinations of brushes, colours and canvas editing options, this program allows full freedom and creativity only limited by the ability of the user.
