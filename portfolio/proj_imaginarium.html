Imaginarium

The Imaginarium is an interactive, immersive experience that runs concurrently on a 3D web application and on-site. Users are guided to reply to a prompt and see how their replies affect the background and shape the experience in real time through a sentiment analysis api.

Hosted Online, The Imaginarium allows users to access and contribute from anywhere in the world. Each submission is stored in a database and fed into the immersive scene. The imaginarium is created using HTML, CSS, Javascript with Three.js for web 3d and PostgreSQL for database and backend handling.
